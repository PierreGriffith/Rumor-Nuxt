A) How to Use - top

In order to use SVG POP Watch, you need to add an element as follows to your HTML: 

	<‍svg id="target" viewBox="0 0 256 256"><‍/svg>

The viewBox property is importante since it defines the part of the svg that will be rendered.

You will also need to import our code by adding the following line to the end of the body of your HTML page. 

	<‍script type="text/javascript" src="js/svgpopwatch-min.js"><‍/script>

Then, you need to add the following code to your javascript file, with your settings: 

	var popWatch = new SVGPopWatch({
	    /* your settings */
	});

B) How to Customize - top

A compreensive list of the available options can be found in the table below

JSON Options
Option Type Description
targetElementId string The ID of the svg element where you want to insert the watch.
clockBasePath string The path to the clock bases.
clockPointersPath string The path to the clock pointers.
weatherPath string The path to the weather icons.
clockStyle object The style you want to give to the clock. For more details check the Clock Style Table
pointerStyle object The style you want to give to the pointers. For more details check the Pointer Style Table
weatherStyle object The style you want to give to the weather icons. For more details check the Weather Style Table
updateInterval integer The time between two consecutive updated to the watch time.
displaySeconds boolean Wether you want to display the seconds pointer or not.
info object Aditional information you want the watch to show. For more details check the Info Table
weatherLocation object The location where you want to check the weather. For more details check the Weather Location Table
customDate date The date and time you want the watch to start counting time.
Clock Style Options
Option Type Description
watchFace string The filename of the watch base, including the extension.
fill string The fill color of the watch base.
fillOpacity number The opacity of the watch base.
stroke string The stroke color of the watch base.
strokeOpacity number The opacity of the watch stroke.
strokeWidth number The width of the watch stroke.
Pointer Style Options
Option Type Description
pointers string The filename of the pointers you want to use, including the extension.
hours object The style of the hours pointer, you can set fill, fillOpacity, stroke, strokeOpacity and strokeWidth. Check the Clock Style Table for more details
minutes object The style of the minutes pointer, you can set fill, fillOpacity, stroke, strokeOpacity and strokeWidth. Check the Clock Style Table for more details
seconds object The style of the seconds pointer, you can set fill, fillOpacity, stroke, strokeOpacity and strokeWidth. Check the Clock Style Table for more details
Weather Style Options
Option Type Description
style string The style that is applied to the weather icons. There are three options: default, monochrome and stroke-only.
color string The fill and/or stroke color of the weather icons, when using monochrome or stroke-only styling
opacity number The opacity of the weather icons.
Info Options
Option Type Description
weather boolean Wether to show weather or not.
Weather Location Options
Option Type Description
city string The city you want to check the weather.
country string The code of the country where the city is. E.g. "us".

C) How to Create Your Custom Assets - top

In order to create your own assets you need to create at least 2 svg files: one for the clock and labels, and another for the pointers.

The document should have the dimension of 256px by 256px. The pointers will rotate around the center of the document.

Ids of svg elements
id Element
clock-zone The watch face itself, without labels. It is expected to be a circle.
clock-labels The labels of the watch. It is optional, but expected to be a group if present.
hours The hours pointer.
minutes The minutes pointer.
seconds The seconds pointer.
weather The weather element. It is expected to be a group.

If you want to create your own weather icons, you need to make sure that the root element has an id of "weather". Check the following table to know how to name your icons.
